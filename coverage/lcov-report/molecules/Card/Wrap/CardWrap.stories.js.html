const docs = `
A very basic Card component with base functionality. Can be extended and used to build other Card types.
Also accepts class attribute so it can be styles with the classes available in the patterns library.
 
## __Options__
To change the style and appearance of the Card use the inputs in the 'Knobs' tab.
 
### Tag
Defines the tag that the card should use, defaults to DIV
- \`div\`
 
### Shadow
Adds box shadow to the Card
- \`m-card-wrap--shadow\`
 
### Rounded
Adds rounded corners to the Card
- \`m-card-wrap--rounded\`
 
### Dashed
Adds dashed border to the Card.
- \`m-card-wrap--dashed\`
 
### Padding Size
Controls the padding of the Card. Available options: xs, sm, md, lg, xl, xxl.
By default there is no padding set.
- \`m-card-wrap--padding-xs"\`
 
### Centered
Centers the content of the Card. Default is top left.
- \`m-card-wrap--centered\`
 
### Content orientation
Sets the layout of the content, using flex-direction:
- \`vertical\`
- \`horizontal\`
- \`vertical-mob\` - Horizontal at all other views
- \`horizontal-mob\` - vertical at all other views
 
## Context of use
For modal inner part, combined with an overlay, for base for building various card components, alerts, notification wrappers.
`
